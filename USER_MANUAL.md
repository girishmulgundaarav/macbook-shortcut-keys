## 1. Core Features & Overview

The macOS Keyboard Shortcuts Guide is built to make learning key combinations visual and interactive:
* **Interactive MacBook Keyboard Layout**: Visualizes key combinations dynamically in real time.
* **Smart Search**: Quickly parses shortcuts by descriptions, key names, or combinations.
* **Category Organization**: Groups shortcuts by functional areas (System, Finder, Navigation, Windows, Safari, Developer).
* **Favorites System**: Save your most-used shortcuts for quick access.
* **Fluid Theme Engine**: Toggle between light and dark modes according to your preference.

## 2. Using the Interactive Virtual Keyboard

The virtual keyboard is the central hub of the application:

### Hover to Highlight Keys
When browsing the shortcuts list:
1. Move your cursor over any shortcut card.
2. The specific keys required for that shortcut will immediately glow and highlight on the virtual keyboard layout.
3. Move your cursor away to clear the highlighting.

### Click Modifiers to Filter
You can narrow down shortcuts based on specific modifier keys:
1. Click any modifier key on the virtual keyboard layout:
   * **⌘ Command**
   * **⌥ Option**
   * **⌃ Control**
   * **⇧ Shift**
   * **fn (Function / Globe)**
2. The keyboard will lock the modifier into an active filtering state (highlighted in blue).
3. The shortcuts list will instantly update to show only shortcuts that include all selected modifier keys.
4. To remove a modifier filter, click that modifier key again on the virtual keyboard, or click **Reset Keys** / **Clear All Filters** in the interface.

## 3. Search and Filters

Locate specific shortcuts with the powerful real-time search:

### Global Search Bar
* Type a search query in the search bar (e.g. `copy`, `quit`, `command c`, or `option`).
* The engine matches queries against shortcut descriptions, category names, and key names/labels.
* Click the **✕** button at the right of the search input to instantly clear the search text.

### Search Hotkeys
Keep your hands on the keyboard with these useful keyboard hotkeys built into the app:
* Press the `/` key from anywhere on the page to automatically place your cursor focus inside the search input.
* Press the `Escape (Esc)` key to clear the current search query, deactivate modifier filters, and unfocus the search bar.

## 4. Categories and Favorite Shortcuts

### Browsing Categories
Use the sidebar menu to view shortcuts by specific macOS sections:
* **All Shortcuts**: Displays the entire collection.
* **Favorites**: Displays only your starred shortcuts.
* **Category Tabs**: Filter by groups such as System, Finder, Navigation, Window Management, Safari, and Developer.
* The badge counter next to each category dynamically updates based on active modifier filters.

### Managing Favorites
You can curate your own custom cheat sheet using the Favorites feature:
* Click the **★ (Star)** icon in the top right of any shortcut card to add it to your favorites.
* The star will turn yellow, indicating it is saved.
* Click the star icon again to remove the shortcut from your favorites list.
* Favorites are saved in your browser's **LocalStorage**, meaning they persist even if you reload the page or close the browser.

## 5. Theme and Personalization

* **Dark and Light Modes**: Click the theme icon (🌙 / ☀️) in the top-right corner of the header to toggle between light and dark themes. The application will also automatically default to your system's preferred theme on first load.
* **Saved Preferences**: Your theme setting is remembered across sessions.
